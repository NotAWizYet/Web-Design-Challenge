Project Summary

This project examines and explores weather data for more than 500 cities as a function of their latitude location. The analysis includes analysis of wind speed, Maximum Temperature, Cloud Index and Humidity. These data are then analyzed and plotted as a function of latitude. The data was drawn from Open Weather Map's website Open Weather Map using Python API calls to that site, then stored in CSV files and integrated into Panda Dataframes and plotted with Matplotlib.
