Project Summary




Example Plot: Max Temperature vs Latitude

Visualizations

This project examines and explores weather data for more than 500 cities as a function of their latitude location. The analysis includes analysis of wind speed, Maximum Temperature, Cloud Index and Humidity. These data are then analyzed and plotted as a function of latitude. The data was drawn from Open Weather Map's website Open Weather Map using Python API calls to that site, then stored in CSV files and integrated into Panda Dataframes and plotted with Matplotlib

Besides the Matplotlib charting, statistical analysis was also conducted on the weather parameters versus the city latitude. At this time the analysis is limited to linear regression with correlation coefficents computed and best fit lines plotted. However we could have applied non-linear models as well or attemted to correlate weather parameters to see if there was a relationship. For the purpose of this study, the aim in this project is limited to providing simple statistical analysis and plots as a foundation for improving the data visualization skills and bringing them to life for in a website platform so that data is available to those on the internet.
