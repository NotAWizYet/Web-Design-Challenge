This is a chart of Max Temperature versus City Latitude. Note the increase in the Max Temperature as the latitude approaches zero (the Equator) The cities selected in for this analysis had specific wind speeds, humidity, temperature and cloudiness recorded for each. Matplotlib and Pandas dataframes were used to assist in the analysis and data was extracted using an API call to Open Weather Map.
